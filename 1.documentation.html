Making the website

Assignment 1 is to build an individual website with html and document our work. I began working on the assignment by looking at html5 templets on the website: https://html5up.net/ and there I found a templet that I liked. Next I watched videos that Hafliði had put up on the website for our course. I then downloaded Brackets to use as a text editor for my website. I have taken a Web Programming course in the past so I didn't need to spend much time on learning how html and Brackets works. I began working on making the header. I needed them all to work together and everypage to look the same. I put up a layout for every page and put up appropriate pictures. Putting up the banner and choosing the layout took about 2 hours. I used the website www.w3schools.com to help me get the layout that I wanted. Then I began writing the text for everypage, if I needed help I either used Google or the w3school.

Uploading on GitHub

First we doawnload Git on: https://git-scm.com/downloads. Next we create an account on github.com. Then we create a new repository, make it public and hatch where it says: Initialize this repository with a README and then create our repository. Next we upload our files and save. Finally we press the setting button and scroll down to GitHub Pages and choose master branch in source, after that the website gives you a link to your website.
